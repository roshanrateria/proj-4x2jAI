## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/roshanrateria/proj-4x2jAI.git
   cd proj-4x2jAI
   ```

2. **Install dependencies**
   ```bash
   npm install .
   ```
3. **Start development server**
   ```bash
   npm run dev
   ```
